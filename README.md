#  WeatherPro - Prognoza Pogody

Responsywna aplikacja webowa do sprawdzania aktualnej pogody oraz prognozy na 5 dni dla wybranego miasta. Projekt zespołowy stworzony w ramach zaliczenia przedmiotu "Wprowadzenie do Technologii Internetowych".

##  Technologie
- **HTML5** (Semantyczna struktura)
- **CSS3** (Mobile First, Flexbox, CSS Grid, Zmienne CSS)
- **Vanilla JavaScript** (ES6+, DOM Manipulation, Async/Await)
- **OpenWeatherMap API** (Zewnętrzne API do pobierania danych pogodowych)

##  Funkcjonalności
- [x] Wyszukiwanie aktualnej pogody po nazwie miasta.
- [x] Walidacja formularza wyszukiwania (RegExp - min. 3 znaki, brak znaków specjalnych).
- [x] Prognoza pogody na najbliższe 5 dni.
- [x] Zmiana jednostek (Celsjusz / Fahrenheit) w locie.
- [x] Geolokalizacja - pobieranie pogody dla aktualnej lokalizacji użytkownika.
- [x] Historia wyszukiwań z zapisem w `localStorage` (zapamiętywanie preferencji).
- [x] Dynamiczna zmiana tła aplikacji w zależności od panujących warunków pogodowych.
